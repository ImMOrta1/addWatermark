# Watermark generator by D.I.M.A Unity

## How to install the project

[Ruby](https://www.ruby-lang.org/en/) is required

1. git clone https://github.com/ImMOrta1/addWatermark.git addWatermark
2. cd addWatermark
3. bower install
4. npm install
5. gem install sass
6. gem install compass
5. gulp
